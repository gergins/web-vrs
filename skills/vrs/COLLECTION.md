# VRS Skill Collection

## Purpose
Provide reusable domain skills for a SIP/WebRTC Video Relay Service platform.

## Catalog Scope
This collection is catalog-only by default.
No runtime behavior is enabled by adding these files.

## Skills
- `realtime-communication`: low-latency media architecture and scaling guidance
- `webrtc-ops`: production RTC operations, diagnostics, and failover
- `interpreter-routing`: queueing, matching, and session handoff strategy
- `accessibility-ux`: Deaf-first product and interaction design
- `compliance-security`: regulatory, privacy, and audit architecture
- `ai-assistive`: assistive AI integration with human-first guardrails
- `incident-response`: operational response and recovery playbooks

## Usage
Use each sub-skill independently based on task scope.
Combine only the minimum set needed for a turn.

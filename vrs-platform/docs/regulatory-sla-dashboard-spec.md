# Regulatory and SLA Dashboard Spec

## Dashboard Audiences
- Operations teams.
- Regulators and procurement reviewers.
- Compliance and audit teams.

## Core SLA Metrics
- Uptime percentage.
- Call success rate.
- Call setup latency.
- Median and p95 media latency.

## Accessibility Metrics
- Interpreter availability rate.
- Queue wait time.
- Video quality score (hands and face clarity proxy).

## Compliance Metrics
- Consent log coverage.
- Data retention policy compliance.
- Audit event completeness.

## Data Sources
- Signaling session events.
- Context/Redis session state snapshots.
- AI policy and ethics audit events.
- Workforce forecasting and assignment events.

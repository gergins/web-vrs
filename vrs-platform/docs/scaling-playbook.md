# Scaling Playbook

## Scale Triggers
- CPU and memory threshold breaches.
- Active session count thresholds.
- Queue length and interpreter wait-time thresholds.

## Horizontal Scaling Actions
- Add signaling replicas.
- Scale TURN capacity and regional relay endpoints.
- Increase assignment/context service replicas.

## Workforce Scaling Actions
- Trigger shortage alerts.
- Invoke standby interpreter pools.
- Prioritize emergency and critical call categories.

## Capacity Planning
- Peak-hour traffic modeling.
- Event surge scenario simulation.
- Periodic stress tests and recovery drills.

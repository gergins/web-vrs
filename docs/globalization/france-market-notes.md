# France Market Notes (Public Sources)

## Scope
Public-source input notes for creating and maintaining `config/country-profiles/fr-france.json`.

## Confirmed Public Signals
1. Emergency accessibility path exists via `114` for Deaf and hard-of-hearing users.
2. French public-sector digital accessibility baseline references RGAA.
3. Telecom regulator ARCEP publishes accessibility-related guidance/expectations.
4. CNIL provides privacy/data-protection governance context for services handling personal data.

## Product Implications
1. Emergency workflow must include country-specific adapter for `114`.
2. Accessibility profile must support RGAA-aligned UX and content controls.
3. Reporting and governance should be regulator-audit ready.
4. Data/consent handling must remain GDPR + French privacy compatible.

## Confidence and Limits

- Final launch requires legal/regulatory review in-country.

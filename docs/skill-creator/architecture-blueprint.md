# Skill Creator Architecture Blueprint

## Layers
1. Experience Layer (apps)
2. Service Layer (APIs)
3. Workflow/Skill Engine
4. Knowledge + Policy Layer
5. Context + Memory Layer
6. Data + Infrastructure Layer

## Core Engines
- Skill registry and resolver
- Workflow orchestrator
- Policy/rules engine
- Context integrity service
- Observability and decision log pipeline

## Required Platform Primitives
- contract-first APIs/events
- capability registry
- skill lifecycle versioning
- human-in-the-loop approvals
- simulation runtime
